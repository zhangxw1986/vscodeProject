13.4复合类型

复合类型是由多个基本数据类型（也可以是复合类型）组成的数据体。JavaScript中的复合类型大致上有如下三种:

  • Object:对象
  • Array:数组
  • Function：函数

13.4.1对象

对象是一系列命名变量、函数的集合。其中命名变量既可以是基本数据类型，也可以是复合类型。对象中的命名变量称为属性，对象中的 函数称为方法。对象访问属性和函数的方法都是通过"."实现的，例如如下代码用于判断浏览器版本：

  • String：字符串类。

13.4.2 数组

数组是一系列的变量。与其他强类型语言不同的是：JavaScript数组元素的类型不必相同。

定义数组的方法有三个：
  • var a = [3,4 ,"3"];
  • var b = [];
  • var c = new Array();

JavaScript为数组提供了一个length属性，该属性可以得到数组的长度，JavaScript的数组长度可以随意变化，它总是等于 所有元素索引最大值+1。

JavaScript作为动态、弱类型语言，归纳起来数组有如下三个特征：

  • JavaScript数组的长度可变，数组长度总等于所有元素索引最大值+1.
  • 同一数组中的元素类型可以不同。
  • 访问数组元素时不会越界，访问并未赋值的数组元素时，该元素的值为undefined。


13.4.3 函数

从语法定义角度看，JavaScript函数与java方法有些相似，但实际上他们差别很大，主要存在如下4点区别：

  • JavaScript函数无需声明返回值。
  • JavaScript函数无需声明形参类型。
  • JavaScript的函数可以独立存在，无需属于任何类。
  • JavaScript函数必须用function关键字定义。

关于typeof和instanceof
